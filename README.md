# Organisation de fichiers en Python

## Description

Ce script permet de ranger automatiquement des fichiers dans des dossiers
selon leur extension (ex: images, documents, vidéos,etc...).

## Fonctionnement

* Le script parcourt un dossier
* Il détecte le type de chaque fichier
* Il crée le dossier adéquat pour chaque type
* Il déplace les fichiers dans ce dernier

## Utilisation

1. Modifier le chemin du dossier à organiser
2. Lancer le script avec :
   python main.py

## Technologies

* Python 3
